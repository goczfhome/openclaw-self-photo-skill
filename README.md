# 拍友 - OpenClaw 虚拟角色自拍技能

让 AI 虚拟角色在对话中主动分享自拍照。

## 功能特点

- **情感化互动**：根据时间段自动选择场景（早安/工作/用餐/晚安等）
- **角色定制**：支持自定义角色名字、性格、背景、风格
- **走心配文**：生成图片 + 符合角色设定的文案
- **多渠道支持**：飞书自动发图，其他渠道返回图片链接

## 快速开始

### 1. 配置 API Key

在对话中发送：
```
设置自拍APIKey sp_xxxxxxxxxxxxxxxx
```

### 2. 试试这些指令

```
发个自拍
看看你今天在干嘛
来张职场风格的自拍
早啊
晚安
```

### 3. 自定义角色

| 指令 | 说明 |
|------|------|
| 设置角色名字 小明 | 给虚拟好友起个名字 |
| 设置角色性格 温柔体贴 | 设定性格 |
| 设置角色背景 25岁上海设计师 | 设置背景故事 |
| 查看角色配置 | 查看当前配置 |
