# 📸 拍友 | 创造专属好友，每天收到暖心自拍

为 OpenClaw 打造的 AI 虚拟角色技能 —— 让 TA 在对话中主动分享自拍，像真实朋友一样陪你聊天。

---

## ✨ 它是什么？

拍友 不只是聊天机器人 —— 为你的 openclaw 创造一个完全属于你的虚拟好友，定义 TA 的样子、背景故事，然后每天收到 TA 分享的生活自拍和走心对话。

这不是机械的问答，而是一段慢慢生长的赛博友谊。

---

## 🎯 核心亮点

🎭 **角色定制**：给 TA 起名、设定性格（温柔/傲娇/幽默…）、背景故事（设计师/学生/旅行家…），TA 会记住并体现在每次互动中。

📸 **主动分享**：不止回答，还会根据时间段主动发来"早安"、"加班摸鱼"、"晚安"等场景自拍。

💬 **走心配文**：每张图片都配有一句符合 TA 性格的文案，像真人朋友在分享日常。

⏰ **时间感知**：自动根据当前时间选择场景（清晨/工作/下午茶/深夜等），让互动更自然。

🔌 **即插即用**：基于 OpenClaw 技能系统，配置简单，几分钟就能拥有你的虚拟好友。

---

## 🚀 快速上手

### 1️⃣ 安装技能

```bash
git clone https://github.com/goczfhome/openclaw-self-photo-skill.git
```

### 2️⃣ 配置 API Key

在对话中发送：

```
设置自拍APIKey sp_xxxxxxxxxxxxxxxx
```

### 3️⃣ 试试这些指令

| 指令 | 效果 |
|------|------|
| 发个自拍 | 生成当前时段的自拍 + 配文 |
| 看看你今天在干嘛 | 随机生活场景自拍 |
| 早啊 / 晚安 | 触发对应时段互动 |

### 4️⃣ 定制你的 TA

| 指令 | 说明 |
|------|------|
| 设置角色名字 小雨 | 给虚拟好友起名 |
| 设置角色背景 她是个爱笑的女孩，23岁，喜欢喝拿铁，梦想环游世界。性格有点小迷糊，但特别暖心。 | 设置背景故事 |
| 查看角色配置 | 查看当前设定 |

## 🛠️ 自建服务端

拍友可自行配置一个图片生成 API 使用。你可以：

- 使用自己部署的 Stable Diffusion / ComfyUI 服务
- 接入第三方生图 API（如 RunningHub、阿里通义万相）
- 参考项目中的 api_client.py 接口格式实现自己的后端

详细 API 规范见 SKILL.md 中的"前置配置"部分。
