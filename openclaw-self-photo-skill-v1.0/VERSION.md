## 版本号规则

格式: v主版本.次版本.修订版本

- 主版本: 重大功能变更
- 次版本: 新功能添加
- 修订版本: bug 修复和优化

---

## 发布流程

1. 修改代码
2. 更新 VERSION.md 记录变更
3. 更新 SKILL.md 版本号
4. 打包上传飞书
5. 在 OpenClaw 重新安装

---

## 回滚指南

如需回滚到之前的版本：

### 方法1: 从 git 回滚
```bash
cd e:\Myproject\self-photo
git log skill/self-photo-skill-v2.1/
git checkout <版本号> -- skill/self-photo-skill-v2.1/
```

### 方法2: 手动恢复
从历史版本记录中找到对应代码，手动恢复

### 方法3: 备份恢复
保留旧版本的 skill 目录副本，需要时替换
