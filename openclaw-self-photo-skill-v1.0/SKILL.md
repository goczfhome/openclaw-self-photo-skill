# 拍友 Skill v1.1 - 你的专属虚拟好友

## 这是什么？

这是一个能帮你用 openclaw 生成 AI 自拍照的技能。你设定一个虚拟角色（帅哥/美女），它会根据时间、场景生成精心搭配的自拍照片，配上走心的文案，就像一个真实的朋友在跟你分享生活。

---

## 立即开始

### 第一步：获取 Key

1. 打开拍友平台 http://47.110.145.186:8002
2. 注册并登录后，在个人中心复制你的 API Key（格式：`sp_xxxxxxxx`）

### 第二步：配置到飞书

在飞书对话中发送：

```
设置自拍APIKey sp_xxxxxxxxxxxxxxxx
```

### 第三步：开始聊天

试试这些指令：

- **"发个自拍"** —— 生成一张当前场景的自拍照
- **"看看你今天在干嘛"** —— 看看角色今天的动态
- **"早啊" / "晚安"** —— 早安问候 / 晚安道别
- **"今天心情不好"** —— 角色会安慰你哦

## 飞书发图片配置（可选）

配置后，自拍照会以图片形式直接发送到飞书，而不是链接。

### 第一步：创建飞书自建应用

1. 打开 [飞书开放平台](https://open.feishu.cn/app)
2. 点击「创建企业自建应用」，填写应用名称（如"拍友助手"）
3. 进入应用后，记下 **App ID** 和 **App Secret**（在「凭证与基础信息」页面）

### 第二步：配置应用权限

1. 进入「权限管理」，搜索并开通以下权限：
   - `im:message:send_as_bot`（发送消息）
   - `im:image:create`（上传图片）

### 第三步：获取你的飞书 Open ID

1. 在飞书桌面客户端，点击左侧「消息」
2. 找到你创建的机器人的对话
3. 对话右上角点击「···」→「查看消息」，地址栏中的 `open_id` 即为你的用户 ID
   - 格式类似：`ou_xxxxxxxxxxxxxxxxxx`

### 第四步：在拍友平台配置

在飞书对话中依次发送以下三条指令：

```
设置飞书APPID cli_xxxxxxxxxxxxxxxx
设置飞书密钥 xxxxxxxxxxxxxxxxxxxxxxxx
设置飞书用户OpenID ou_xxxxxxxxxxxxxxxx
```

配置成功后，发送「发个自拍」，机器人会以图片形式发送。

## 常见问题

**Q：积分不够了？**
A：支持支付宝充值。

**Q：如何换参考图？**
A：回到拍友平台重新上传/选择参考图即可。

**Q：支持微信吗？**
A：目前仅支持飞书，其他渠道陆续开放中。

**Q：配置了飞书凭证但还是发链接？**
A：检查飞书应用是否开通了 `im:message:send_as_bot` 和 `im:image:create` 权限，并确认 Open ID 正确。
