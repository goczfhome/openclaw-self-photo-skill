# 拍友 Skill v1.0 - 为你的open打造专属虚拟好友

## 这是什么？

这是一个能帮你用 openclaw 生成 AI 自拍照的技能。你设定一个虚拟角色（帅哥/美女），它会根据时间、场景生成精心搭配的自拍照片，配上走心的文案，就像一个真实的朋友在跟你分享生活。

**支持飞书和微信双平台配置。**

---

## 立即开始

### 第一步：获取 Key

1. 打开拍友平台 http://47.110.145.186:8002/
2. 邮箱免费注册会员
3. 上传一张图片创建虚拟角色
4. 点击生成APIKey sp_xxxxxxxxxxxxxxxx

### 第二步：配置 API 地址（重要！）

**支持飞书和微信**，请在对应平台发送以下配置命令：

**飞书配置：**
```
设置自拍API地址 http://47.110.145.186:8002
设置自拍APIKey sp_xxxxxxxxxxxxxxxx
```

**微信配置：**
```
设置自拍API地址 http://47.110.145.186:8002
设置自拍APIKey sp_xxxxxxxxxxxxxxxx
```

> 注意：必须先配置 API 地址，再配置 API Key，微信才能正常使用！

### 第三步：开始聊天

试试这些指令：

- **"发个自拍"** —— 生成一张当前场景的自拍照
- **"看看你今天在干嘛"** —— 看看角色今天的动态
- **"早啊" / "晚安"** —— 早安问候 / 晚安道别

## 常见问题

**Q：积分不够了？**
A：支持支付宝充值。

**Q：如何换参考图？**
A：回到拍友平台重新上传/选择参考图即可。

**Q：支持微信吗？**
A：支持！飞书和微信都支持，配置方法见上方第二步。

**Q：微信配置了但不生效？**
A：必须先发送"设置自拍API地址 http://47.110.145.186:8002"，再发送"设置自拍APIKey"，顺序不能颠倒。
