# 拍友 Skill - 你的专属虚拟好友

## 这是什么？

这是一个能帮你用 openclaw 生成 AI 自拍照的技能。你设定一个虚拟角色（帅哥/美女/宠物），它会根据时间、场景生成精心搭配的自拍照片，配上走心的文案，就像一个真实的朋友在跟你分享生活。

---

## 立即开始

### 第一步：获取 Key

1. 打开拍友平台

### 第二步：配置到飞书

在飞书对话中发送：

```
设置自拍APIKey sp_xxxxxxxxxxxxxxxx
```

### 第三步：开始聊天

试试这些指令：

- **"发个自拍"** —— 生成一张当前场景的自拍照
- **"看看你今天在干嘛"** —— 看看角色今天的动态
- **"早啊" / "晚安"** —— 早安问候 / 晚安道别
- **"今天心情不好"** —— 角色会安慰你哦

---
